# Python Fundamentals (MOD_01)

## 📋 Description

MOD_01 introduces core Python concepts through beginner-friendly garden and plant-themed exercises. Learn variables, data types, string formatting, input/output, and basic program structure.

## ✨ Features

- **Variables & Data Types** - Integers, floats, strings, and booleans
- **String Formatting** - F-strings and basic text manipulation
- **Input/Output** - Print statements and user interaction
- **Basic Logic** - Conditionals and comparisons
- **Program Structure** - Functions and main entry points
- **Type Hints** - Introduction to type annotations

## 🏗️ Architecture

```
Simple Function-Based Programs
    ├── Variables & Constants
    ├── String Formatting
    ├── Basic I/O
    └── Simple Data Processing
```

## 🛠️ Tech Stack

- **Python**: 3.10+
- **Standard Library**: Built-in functions only

## 🚢 Deployment

All exercises are standalone scripts - simply run with Python:

```bash
python ex0/ft_garden_intro.py
```

## 🔧 Troubleshooting

| Issue | Solution |
|-------|----------|
| `python: command not found` | Install Python 3.10+ or use `python3` |
| Syntax errors | Check indentation and quotes match |
| Variable not defined | Ensure variable is declared before use |
